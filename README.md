# Endline
Python script to change file line endings

### Help
```
usage: endline.py [-h] [-v] [-o <output-file>] [-u] [-w] [-m] <input-file>

Change file line ending (LF, CRLF, or CR).

positional arguments:
  <input-file>      input file

optional arguments:
  -h, --help        show this help message and exit
  -v, --version     show program's version number and exit
  -o <output-file>  optional output file
  -u, --unix        use unix style (LF) line endings
  -w, --windows     use windows style (CRLF) line endings
  -m, --mac         use mac style (CR) line endings
```
